# 🌱 数字花园 - 最终版

一个结合数学学习与游戏化种植的纯HTML网页游戏！通过解答数学题来解锁植物，建造属于你的美丽花园。

## ✨ 功能特点

- 🧮 **三种难度的数学题**：简单（加减法）、中等（混合运算、乘法）、困难（复杂运算、除法）
- 🌻 **6种可爱的植物**：向日葵、郁金香、玫瑰、樱花、仙人掌、大树
- 🌱 **植物生长系统**：植物有多个生长阶段，通过答题施肥让它们长大
- 📚 **错题库功能**：自动记录错题，支持筛选、练习、导出
- 📖 **植物图鉴系统**：查看已解锁和未解锁的植物，显示收集进度
- 💾 **自动保存进度**：使用localStorage保存游戏数据，刷新不丢失
- 🎨 **美观的界面**：渐变色设计，流畅的动画效果
- 📱 **响应式布局**：支持手机和电脑浏览器

## 🚀 如何运行

### 方法1：直接打开（最简单）
1. 双击 `index-ultimate.html` 文件

## 🎮 游戏说明

### 快速开始
1. **种植**：点击花园空位，选择向日葵种上
2. **答题**：点击"做数学题"，答对解锁更多植物
3. **收集**：解锁全部6种植物，填满花园！

### 详细玩法

#### 🌱 花园操作
- 点击**空的格子** → 选择已解锁的植物种植
- 点击**已种植物** → 管理/施肥/移除
- 植物生长阶段：🟫 → 🌱 → 🌿 → 最终形态

#### 🧮 答题系统
- 点击**难度按钮** → 切换题目难度（简单/中等/困难）
- 点击**选项按钮** → 选择你的答案
- 答对增加计数，答错自动记录到错题本

#### 📚 错题库
- **筛选**：按难度查看错题
- **练习**：重做错题，巩固知识
- **导出**：将错题导出为txt文件保存
- **移除**：从错题本删除已掌握的题目

#### 📖 植物图鉴
- 查看所有植物及其解锁条件
- 彩色卡片 = 已解锁，灰色卡片 = 未解锁
- 显示收集进度百分比

### 解锁条件

| 植物 | 解锁条件 | 生长阶段 |
|------|----------|----------|
| 🌻 向日葵 | 初始解锁 | 3阶段 |
| 🌷 郁金香 | 答对3题 | 3阶段 |
| 🌹 玫瑰 | 答对8题 | 4阶段 |
| 🌸 樱花 | 答对15题 | 4阶段 |
| 🌵 仙人掌 | 答对25题 | 3阶段 |
| 🌳 大树 | 答对40题 | 5阶段 |

## 🔧 自定义修改

### 添加新植物
在 `PLANTS` 数组中添加新对象：
```javascript
{ 
    id: 'newplant', 
    name: '新植物', 
    emoji: '🌺', 
    stages: 3, 
    unlockAt: 50, 
    desc: '描述文字' 
}
```

### 修改题目难度
修改 `generateEasyQuestion`、`generateMediumQuestion`、`generateHardQuestion` 函数中的数值范围。

### 更换主题颜色
修改CSS中的颜色值，如 `#22c55e`（绿色）、`#3b82f6`（蓝色）等。

## ❓ 常见问题

**Q：游戏进度保存在哪里？**
A：保存在浏览器的localStorage中，键名为 `digitalGardenState_final`。

**Q：如何清空游戏进度？**
A：在浏览器控制台执行：
```javascript
localStorage.removeItem('digitalGardenState_final')
```
然后刷新页面。

**Q：可以在手机上玩吗？**
A：可以！游戏支持响应式布局，手机浏览器也能完美运行。

**Q：如何分享给我的朋友？**
A：直接发送 `index-ultimate.html` 文件，对方用浏览器打开即可。
