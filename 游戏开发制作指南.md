## 🎯 项目概述

### 什么是数字花园？
数字花园是一款结合数学学习和种植养成的网页小游戏。玩家通过解答数学题来解锁植物，并在花园中种植它们。

### 游戏特点
- ✨ 完全在浏览器中运行，无需安装
- 📊 自动保存进度到浏览器本地存储
- 📚 错题库功能，复习错题
- 📖 植物图鉴，展示收集进度
- 🎨 美观的UI，流畅的交互
- 🎯 三种难度：简单、中等、困难

### 游戏截图说明
游戏界面包含：
- **顶部统计区**：显示答对、答错、解锁植物、错题数量
- **解锁进度条**：显示已解锁植物和下一个解锁目标
- **花园区域**：4×3的格子，可种植植物
- **操作按钮**：做数学题、查看错题库、刷新花园

## 💻 技术选型详解

### 为什么选择纯HTML/CSS/JavaScript？

| 技术 | 作用 | 为什么选择它 |
|------|------|--------------|
| HTML | 结构框架 | 所有网页的基础，浏览器原生支持 |
| CSS | 样式和美化 | 让界面好看，交互流畅 |
| JavaScript | 游戏逻辑 | 实现所有功能：题目、种植、保存等 |
| localStorage | 数据存储 | 浏览器自带的保存功能，不需要服务器 |

### 相比其他方案的优势
- ✅ 无需安装额外软件，浏览器就能运行
- ✅ 学习成本低，这三个是前端基础
- ✅ 跨平台，Windows/Mac/Linux/手机都能玩
- ✅ 文件小，传输快
- ✅ 单文件即可运行，部署简单

### 第一步：创建基础HTML结构

#### 📖 什么是HTML？
HTML（超文本标记语言）是网页的骨架，就像房子的框架。

#### 代码实现
创建一个名为 `index.html` 的文件：

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>🌱 数字花园</title>
    <style>
        /* CSS样式会写在这里 */
    </style>
</head>
<body>
    <!-- 页面内容会写在这里 -->
    <h1>🌱 数字花园</h1>
    
    <script>
        // JavaScript代码会写在这里
    </script>
</body>
</html>
```

#### 代码解释
- `<!DOCTYPE html>`：告诉浏览器这是一个HTML5文件
- `<html>`：整个网页的根标签
- `<head>`：放置网页信息（标题、样式等），不在页面显示
- `<body>`：放置页面内容（文字、图片、按钮等），会显示出来
- `<script>`：放置JavaScript代码

### 第二步：添加基础样式（CSS）

#### 📖 什么是CSS？
CSS（层叠样式表）是网页的化妆师，让页面变得好看。

#### 在 `<style>` 标签中添加：

```css
* {

body {
```

#### CSS基础语法解释
- `*`：选择所有元素，设置基础样式
- `body`：设置页面背景和字体
- `.container`：类选择器，控制主内容宽度居中
- `margin`：外边距（元素外面的空白）
- `padding`：内边距（元素里面的空白）
- `border-radius`：圆角效果

```javascript
function initGarden() {
```

### 第五步：加载和保存进度

```javascript
function loadState() {
    const saved = localStorage.getItem('digitalGardenState');
    if (saved) {
        gameState = JSON.parse(saved);
    } else {
        initGarden();
    }
}

function saveState() {
    localStorage.setItem('digitalGardenState', JSON.stringify(gameState));
}
```

#### 代码解释
- `localStorage`：浏览器提供的本地存储空间，刷新不丢失
- `JSON.parse()`：把文本转成JavaScript对象
- `JSON.stringify()`：把JavaScript对象转成文本

### 第六步：渲染花园界面

```javascript
function renderGarden() {
    const grid = document.getElementById('gardenGrid');
    grid.innerHTML = '';
    
    gameState.garden.forEach(slot => {
        const btn = document.createElement('button');
        btn.className = 'garden-slot' + (slot.plantId ? ' has-plant' : '');
        btn.textContent = getPlantEmoji(slot.plantId, slot.stage);
        btn.onclick = () => openShop(slot.id);
        grid.appendChild(btn);
    });
}

function getPlantEmoji(plantId, stage) {
    const plant = PLANTS.find(p => p.id === plantId);
    if (!plant) return '🟫';
    if (stage <= 0) return '🟫';
    if (stage <= 2) return GROWTH_STAGES[stage - 1];
    return plant.emoji;
}
```

#### 代码解释
- `document.getElementById()`：获取页面上的元素
- `innerHTML`：设置元素的HTML内容
- `createElement()`：创建新的HTML元素
- `appendChild()`：把元素添加到页面
- `find()`：在数组中查找符合条件的元素

```javascript
function generateEasyQuestion() {
    const type = Math.random();
    let a, b, question, answer;
    
    if (type < 0.6) {
        a = Math.floor(Math.random() * 10) + 1;
        b = Math.floor(Math.random() * 10) + 1;
        question = `${a} + ${b} = ?`;
        answer = a + b;
    } else {
        a = Math.floor(Math.random() * 10) + 5;
        b = Math.floor(Math.random() * a) + 1;
        question = `${a} - ${b} = ?`;
        answer = a - b;
    }
    
    const wrong = generateWrongOptions(answer, 'easy');
    return { 
        id: Date.now(), 
        question, 
        answer, 
        options: shuffleArray([answer, ...wrong]), 
        difficulty: 'easy' 
    };
}
```

#### 中等题生成：
```javascript
function generateMediumQuestion() {
    const type = Math.random();
    let a, b, c, question, answer;
    
    if (type < 0.4) {
        a = Math.floor(Math.random() * 30) + 10;
        b = Math.floor(Math.random() * 20) + 5;
        c = Math.floor(Math.random() * 10) + 5;
        question = `${a} + ${b} - ${c} = ?`;
        answer = a + b - c;
    } else if (type < 0.7) {
        a = Math.floor(Math.random() * 12) + 2;
        b = Math.floor(Math.random() * 12) + 2;
        question = `${a} × ${b} = ?`;
        answer = a * b;
    } else {
        a = Math.floor(Math.random() * 100) + 50;
        b = Math.floor(Math.random() * 50) + 10;
        question = `${a} - ${b} = ?`;
        answer = a - b;
    }
    
    const wrong = generateWrongOptions(answer, 'medium');
    return { 
        id: Date.now(), 
        question, 
        answer, 
        options: shuffleArray([answer, ...wrong]), 
        difficulty: 'medium' 
    };
}
```

#### 困难题生成：
```javascript
function generateHardQuestion() {
    const type = Math.random();
    let a, b, c, d, question, answer;
    
    if (type < 0.25) {
        a = Math.floor(Math.random() * 50) + 20;
        b = Math.floor(Math.random() * 30) + 10;
        c = Math.floor(Math.random() * 20) + 5;
        d = Math.floor(Math.random() * 10) + 1;
        question = `${a} + ${b} - ${c} + ${d} = ?`;
        answer = a + b - c + d;
    } else if (type < 0.5) {
        a = Math.floor(Math.random() * 30) + 10;
        b = Math.floor(Math.random() * 20) + 5;
        c = Math.floor(Math.random() * 10) + 2;
        question = `(${a} - ${b}) × ${c} = ?`;
        answer = (a - b) * c;
    } else if (type < 0.75) {
        a = Math.floor(Math.random() * 15) + 5;
        b = Math.floor(Math.random() * 12) + 3;
        c = Math.floor(Math.random() * 20) + 10;
        question = `${a} × ${b} + ${c} = ?`;
        answer = a * b + c;
    } else {
        b = Math.floor(Math.random() * 12) + 3;
        answer = Math.floor(Math.random() * 15) + 5;
        a = b * answer;
        c = Math.floor(Math.random() * 10) + 1;
        question = `${a} ÷ ${b} + ${c} = ?`;
        answer = Math.floor(a / b) + c;
    }
    
    const wrong = generateWrongOptions(answer, 'hard');
    return { 
        id: Date.now(), 
        question, 
        answer, 
        options: shuffleArray([answer, ...wrong]), 
        difficulty: 'hard' 
    };
}
```

#### 代码解释
- `Math.random()`：生成0到1之间的随机数
- `Math.floor()`：向下取整
- 生成错误选项时，避免和正确答案重复
- `shuffleArray()`：打乱选项顺序

### 模块1：错题库系统

#### 功能说明
- 自动记录错题
- 按难度筛选
- 练习错题
- 导出错题

```javascript
function addToMistakes(question) {
    const exists = gameState.mistakes.find(m => m.question === question.question);
    if (!exists) {
        gameState.mistakes.push({
            id: Date.now(),
            question: question.question,
            answer: question.answer,
            difficulty: question.difficulty,
            wrongAt: new Date().toLocaleString(),
            wrongCount: 1
        });
    } else {
        exists.wrongCount++;
        exists.wrongAt = new Date().toLocaleString();
    }
    saveState();
}

function exportMistakes() {
    if (gameState.mistakes.length === 0) {
        alert('没有错题可导出！');
        return;
    }
    
    const data = gameState.mistakes.map(m => {
        return `【${m.difficulty}】${m.question}\n正确答案：${m.answer}\n错误次数：${m.wrongCount}\n记录时间：${m.wrongAt}\n---`;
    }).join('\n');
    
    const blob = new Blob([`🌱 数字花园错题集\n\n${data}`], { type: 'text/plain' });
    const url = URL.createObjectURL(blob);
    const a = document.createElement('a');
    a.href = url;
    a.download = `错题集_${new Date().toLocaleDateString()}.txt`;
    a.click();
    URL.revokeObjectURL(url);
}
```
